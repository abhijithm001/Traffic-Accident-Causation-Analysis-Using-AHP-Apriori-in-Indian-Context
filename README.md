# Traffic Accident Causation Analysis Using AHP–Apriori

## Overview
This project presents a hybrid AHP–Apriori framework for analysing road accident severity using 15,000 Indian road accident records collected between 2018 and 2023.

## Key Findings

- Vehicle Type Involved was the most influential factor (Weight = 0.3275)
- Lighting Conditions ranked second (Weight = 0.1825)
- Dark Lighting + Dry Road produced a fatality rate of 34.3%
- Stormy Weather + Pedestrian produced the highest fatality rate (36.8%)
- The framework identified hidden multi-factor accident patterns not visible through single-variable analysis
