# Traffic Accident Causation Analysis Using AHP–Apriori in Indian Context

This project presents a hybrid AHP–Apriori framework for analysing road accident severity using 15,000 Indian road accident records collected between 2018 and 2023.

The study replaces traditional subjective AHP pairwise comparisons with statistically derived Cramer's V values, making the ranking process fully objective and reproducible.

The project identifies key accident causation factors, ranks them using Analytic Hierarchy Process (AHP), and discovers multi-factor accident patterns using the Apriori association rule mining algorithm.
